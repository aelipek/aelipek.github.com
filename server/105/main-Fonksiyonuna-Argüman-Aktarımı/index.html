main Fonksiyonuna Argüman Aktarımı

main() fonksiyonunun argüman almadığını belirtmek için void ( main(void) ) yapısını kullanıyorduk. Şimdi ise main() fonsiyonuna nasıl argüman aktarılacağını göstereceğiz.

#include<stdio.h>
int main(int argc,int *argv[]){
int i;
for(i=0;i<argc;i++){
	puts(argv[i]);
}

return 0;
}

örnek kodumuzu deneme.c adıyla masaüstüne kaydediyoruz. Sonra komut satırına

~$ cd Masaüstü  		//masaüsütüne geçiyoruz
~$ gcc deneme.c -o deneme       // gcc programı "deneme" adında çalıştırılabilir bir dosya oluşturacaktır.

Şimdi kodumuza argüman göndererek çalıştırmayı deneyelim.

~$ ./deneme ferhat Akbulut

Çıktımız ;

./deneme
ferhat
akbulut

şeklinde olacaktır. main fonksiyonunun parametreleri int ve char** türlerinden oluşabiliyor. argc parametresi program ismi dahil komut satırı argümanlarının sayısını belirtir. Yani ./deneme, ferhat, Akbulut ifadeleri komut satırı argümanlarımızdır. argv ifademiz ise komut satırı argümanlarını oluşturan yazıların adreslerinin yerleştiği dizinin adresini belirtmektedir. Yani argv bir pointer dır. Başka bir örnek yapalım ;

#include<stdio.h>
void ters_cevir(char katar[]);
int main(int argc,int *argv[]){
	int i;
	for(i=0;i<argc;i++){
	
		ters_cevir(argv[i]);
		printf("\n");
	}
	printf("\n");

	return 0;

}
void ters_cevir(char katar[]){
	int i,uz;
	uz=strlen(katar)-1;
	for(;uz>=0;uz--){
		printf("%c",katar[uz]);
	}
}

Program çalıştığında kendisine verilen argümanları tersten yazacaktır. Programı deneme_2.c adıyla kaydediyoruz.

~$ gcc deneme_2.c -o deneme_2	
~$ ./deneme_2 ferhat Akbulut
